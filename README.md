# My To-Do List Application

Welcome to the repository for my simple To-Do List application! This app helps you manage your daily tasks efficiently.

## Features
* Add new tasks.
* Edit tasks.
* Delete tasks.
* Log in for save tasks
* Sign up
* Search for tasks.
* Simple and clean user interface.
